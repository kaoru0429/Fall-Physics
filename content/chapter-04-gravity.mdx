# Chapter 4: 重力的陷阱 - 萬有引力與自由落體

你剛從隕石迷宮中逃脫，還來不及慶祝，一陣強烈的警報聲就響徹駕駛艙。「銀河快車」被前方一顆巨大行星的引力捕捉，正沿著一條螺旋軌道加速墜落！你試圖啟動引擎逃離，但引擎的推力在巨大的引力面前顯得微不足道。

> 💬 **Physica**: 「這就是宇宙中最普遍的力——萬有引力！牛頓發現，任何兩個有質量的物體之間都存在相互吸引的力。行星的質量越大，距離越近，引力就越強。你現在正體驗著一場標準的『自由落體』運動。」

窗外的景象越來越清晰，你能看到行星的大氣層和一座懸浮在雲端、由光芒構成的古老橋樑。不幸的是，橋樑正在引力的撕扯下逐漸崩塌。你必須在列車墜毀前，利用引力安全地穿越這座橋。

## �� Boss 戰：奇異點暗影

你駕駛列車，像衝浪一樣駕馭著引力波，驚險地穿過了崩塌的光之橋。在橋的盡頭，引力的源頭出現了：一個由純粹重力構成、扭曲光線的黑暗球體——**奇異點暗影**。它沒有實體，但它的引力場卻能撕碎一切靠近的物質。

> 💬 **Physica**: 「那就是 Boss！它本身就是一個微型黑洞。你無法用任何物理攻擊傷害它，任何東西都會被它的引力吞噬。但你注意到嗎？它的引力場正在讓周圍的隕石加速！」

**�� 勝利提示**: 你無法摧毀引力，但你可以利用引力！你需要利用 Boss 的引力來加速，達到逃逸所需的速度。這被稱為「引力彈弓效應」。

**�� 攻略**: 你需要計算出一個完美的切入角度和速度。駕駛「銀河快車」掠過奇異點暗影的事件視界邊緣，但不能被吸進去。在最接近它的一瞬間，啟動你之前獲得的【反作用力推進器】，在引力提供的巨大加速度基礎上再增加一個爆發性的推力。這個疊加的速度將足以讓你擺脫引力束縛，像從彈弓射出的石子一樣被高速拋射出去，逃離險境！

成功逃逸後，你發現列車的導航系統下載了 Boss 的引力數據，你獲得了【引力井產生器】，一個可以短時間內束縛敵人的裝置。

## �� 章節總結

你戰勝了宇宙中最基本的力，恭喜你對引力的理解達到了新的高度！

| 核心知識點 | 定義 | 應用 |
| :--- | :--- | :--- |
| **萬有引力定律** | 宇宙中任何兩個物體都相互吸引，力的大小與質量乘積成正比，與距離平方成反比。 | 行星運行、潮汐現象、蘋果落地。 |
| **自由落體** | 物體只在重力作用下的運動，加速度恆定（在同一地點）。 | 跳水、高空拋物。 |
| **重力加速度 (g)** | 在地球表面約為 9.8 m/s²。 | 物體下落速度每秒增加 9.8 m/s。 |
| **引力彈弓效應** | 利用行星或其他天體的引力來為太空探測器加速。 | 航海家號探測器。 |
